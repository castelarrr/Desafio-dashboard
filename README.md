📊 Processamento e Transformação de Dados com Power BI
📄 Sobre o Projeto
Este repositório contém o projeto prático desenvolvido durante o curso de Power BI da DIO. O objetivo principal foi aplicar técnicas avançadas de ETL (Extract, Transform, Load) utilizando o Power Query para preparar uma base de dados bruta para análises estratégicas e tomadas de decisão.

🎯 Objetivos do Desafio
O desafio consistiu em replicar e aprimorar as etapas de tratamento de dados vistas em aula, garantindo que o modelo de dados fosse:

Otimizado (Alta performance)

Limpo (Sem ruídos ou erros)

Pronto para Visualização (Estrutura relacional correta)

🛠️ Transformações Realizadas (Power Query)
Durante o desenvolvimento, executei as seguintes etapas cruciais de limpeza e modelagem:

Tratamento de Cabeçalhos e Tipos de Dados: Verificação rigorosa e ajuste de colunas numéricas, de texto e de data para garantir a integridade dos cálculos.

Limpeza de Valores Nulos: Identificação e tratamento de registros vazios ou inconsistentes para não enviesar os resultados.

Transformação de Colunas: Divisão de colunas complexas (como endereços) e criação de colunas condicionais para segmentação inteligente.

Mesclagem de Tabelas (Merge): Consolidação estratégica de informações de diferentes fontes (ex: Departamentos e Colaboradores) para enriquecer a profundidade da análise.

Otimização do Modelo: Remoção de colunas desnecessárias, reduzindo o tamanho do arquivo e aumentando a velocidade de processamento do relatório.
