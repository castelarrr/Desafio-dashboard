📊 Processamento e Transformação de Dados com Power BI
Este repositório contém o projeto prático desenvolvido durante o curso de Power BI da DIO. O objetivo principal foi aplicar técnicas de ETL (Extract, Transform, Load) utilizando o Power Query para preparar uma base de dados bruta para análises estratégicas.

🎯 Objetivos do Desafio
O desafio consistiu em replicar e aprimorar as etapas de tratamento de dados vistas em aula, garantindo que o modelo de dados fosse otimizado, limpo e pronto para a criação de visuais.

🛠️ Transformações Realizadas (Power Query)
Durante o desenvolvimento, realizei as seguintes etapas de limpeza e modelagem:

Tratamento de Cabeçalhos e Tipos de Dados: Verificação e ajuste de colunas numéricas, de texto e de data para evitar erros em cálculos futuros.

Limpeza de Valores Nulos: Identificação e tratamento de registros vazios ou inconsistentes.

Transformação de Colunas: Divisão de colunas complexas (como endereços) e criação de colunas condicionais para categorização.

Mesclagem de Tabelas (Merge): Consolidação de informações de diferentes fontes (ex: Departamentos e Colaboradores) para enriquecer a análise.

Otimização do Modelo: Remoção de colunas desnecessárias que não seriam utilizadas no relatório final, garantindo melhor performance.
